Paper and Review Summary. Briefly describe the contributions of the paper to computer vision. Include a concise, bulleted list of the paper's main strengths and a concise, bulleted list of the paper's main weaknesses. Please keep these brief. You will elaborate on the pros/cons in the subsequent text boxes below. The paper presents a method for 3D object reconstruction (voxel occupancy) using aggregated local structured prediction on a single depth map to recover voxel occupancy. The features over the depth map are an offset feature, similar to others often used in local description and a novel feature (spider) which computes a normalized distance between a point and the closest edge in three ways (image distance, surface distance, and normal distance) for a set of angles.

Strengths:
Aggregate local features work well (a novel result)
Survey of the background

Weaknesses:
Simplistic dataset
Structured forest gives minimal gain over k-means
Spider gives minimal gain over local features alone
Hallucinates in the normal direction, but not more.

The strongest point this paper makes, and it is a very strong point, is how well local offset features are at predicting shape. The result outperforms a version of [39] (though the main contribution of [39] was the physical model rather than single object reconstruction) and a baseline "bounding box" approach.

It is also not reasonable to ask for comparisons with unpublished papers and papers published after the CVPR deadline. The main problem with this paper is the simplicity of the dataset. It can be described using a very small vocabulary it's mostly boxes of various aspect (eg. cereal, soap), some general cylinders (eg table leg, pringles), and harder classes which generally have a single plane of symmetry (eg. aunt jemima syrup, arm and hammer detergent, mom to mom juice, windex and similar spray bottles). In all, hard objects make up roughly 20 of the 125 objects, and of the 20 hard objects, many would be reasonably estimated using generalized cylinders. Without a generalized cylinder baseline it's hard to tell whether the performance is actually high quality.

The full structured forest result does not significantly beat a k-means approach (-.004) or a structured forest over just the offset features (-.005). It is not clear if this is because the dataset is overly simple, or the structured forest and spider features are not necessary to achieve good results, a major detractor from the paper.

In figure 9, the authors show qualitative results for data which includes occlusions. While the result is satisfying for parts, the method completely fails to hallucinate occluded voxels that are not directly behind the depthmap.
